## üì¶ Method 1: Edge Add-ons Store (Recommended)

### Step 1: Visit the Store
1. Go to [Keka Time Capture Pro on Edge Add-ons](https://microsoftedge.microsoft.com/addons/detail/keka-time-capture-pro/nafcioiaipfahhkpgmgbjpildbkabonn)
2. You'll see the extension page with description and screenshots

### Step 2: Install
1. Click the **"Get"** button
2. Wait for the download to complete
3. Click **"Add extension"** when prompted

### Step 3: Pin the Extension
1. Click the puzzle icon (üß©) in your browser toolbar
2. Find "Keka Time Capture Pro"
3. Click the pin icon to keep it visible

### Step 4: First Use
1. Navigate to your company's Keka attendance page
2. Click the extension icon
3. The extension will automatically sync your data

‚úÖ **Done!** You're ready to start tracking.

## üì• Method 2: Manual Installation (Developer Mode)

### Prerequisites
- Git installed on your computer
- Basic familiarity with command line

### Step 1: Clone Repository
```bash
git clone https://github.com/sahilkumarsahoo07/Keka-Time-Capture-Pro.git
cd Keka-Time-Capture-Pro
```

### Step 2: Enable Developer Mode
1. Open Edge and go to `edge://extensions/`
2. Toggle **"Developer mode"** (bottom-left corner)

### Step 3: Load Extension
1. Click **"Load unpacked"**
2. Navigate to the cloned folder
3. Select the folder and click **"Select Folder"**

### Step 4: Verify Installation
- You should see "Keka Time Capture Pro" in your extensions list
- The extension icon will appear in your toolbar

## üêõ Troubleshooting Installation Issues

### Extension Won't Install
**Problem**: "Add extension" button grayed out  
**Solution**: Make sure you're using a compatible browser version (Edge 88+)

### Extension Icon Not Showing
**Problem**: Can't find the extension icon  
**Solution**: 
1. Click the puzzle icon (üß©)
2. Find the extension
3. Click the pin icon

### Data Not Loading
**Problem**: Extension opens but shows no data  
**Solution**:
1. Make sure you're on the Keka attendance page
2. Refresh the page
3. Click the extension icon again

### Manual Installation Fails
**Problem**: "Load unpacked" not working  
**Solution**:
1. Verify Developer mode is enabled
2. Make sure you selected the correct folder
3. Check for any error messages in the console
